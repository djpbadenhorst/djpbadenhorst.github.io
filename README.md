# README
Github page

## INSTALL NODE MODULES
To install node modules, run the following:
```
npm install
```

## NPM SCRIPTS
To dev page, run the following:
```
npm run dev
```

To deploy to github pages, run the following:
```
npm run build
npm run deploy
```
